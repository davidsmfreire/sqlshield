# Security Policy

## Supported versions

sqlshield is pre-1.0; only the most recent release receives security fixes.

## Reporting a vulnerability

Please **do not** file a public GitHub issue for security problems.

Email the maintainer at **davidsmfreire@gmail.com** with:

- a description of the issue,
- steps to reproduce (input SQL / schema / source file if applicable),
- your assessment of impact.

You can expect an initial response within 7 days. Once a fix lands, credit will
be given in the release notes unless you prefer to remain anonymous.
